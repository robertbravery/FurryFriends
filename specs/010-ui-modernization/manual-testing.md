# Manual Testing Checklist: Enterprise UI Modernization

## Visual modernization
- [ ] Dashboard shell uses the modern HSL color palette in light and dark themes.
- [ ] Sidebar uses dark high-contrast styling with vector icons and pill active state.
- [ ] Header search bar and profile dropdown use rounded modern styling.
- [ ] Booking wizard displays connected timeline step progress.

## Component behavior
- [ ] StatusPill displays Active, Inactive, Verified, and default states with visible labels.
- [ ] SkeletonLoader displays table and card skeleton layouts while loading.
- [ ] EmptyState displays vector-style placeholder, heading, description, and optional action.
- [ ] Pet Walker details popup shows About, Schedule & Areas, and Reviews tabs.

## Directory pages
- [ ] Client list table has rounded borders, hover transitions, status chips, and vector action buttons.
- [ ] Pet Walker list table has rounded borders, hover transitions, status chips, and vector action buttons.
- [ ] Loading states show skeleton layouts instead of flat text.

## Responsive and accessibility
- [ ] Mobile navigation can be opened and closed.
- [ ] Interactive controls are reachable by keyboard.
- [ ] Empty and error states include visible text.
- [ ] Light and dark theme contrast is acceptable.
